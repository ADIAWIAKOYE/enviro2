Stick anything on your website!

== Description == 
Make ANY element on your website sticky. Use for sidebars, headers, call to actions, the list goes on! We've made the sticky process super simplistic!

<h4>Key points</h4>
<p><strong>Multiple sticky elements on one page</strong>. Target multiple css classes on one page giving you get complete control over your sticky elements.</p>
<p><strong>Stick absolutely anything.</strong> We didn't name the plugin Sticky Anything for no reason!</p>
<p><strong>Responsive Controls.</strong> Control what screen-sizes elements should stick and unstick.</p>
<p><strong>100s of customisability combinations.</strong> Sticky elements can be customised to suit their individual needs.</p>
